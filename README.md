# htmplate

HTML boilerplate with npm scripts including Sass, Autoprefixer, stylelint and Browsersync.

## Get Started

```bash
$ yarn
```
or

```bash
$ npm install
```

## Develop

```bash
$ yarn dev
```

or

```bash
$ npm run dev
```

## Build

```bash
$ yarn build
```

or

```bash
$ npm run build
```

## Commands

```bash
$ yarn run
```

or

```bash
$ npm run
```
